# Tisha Movaliya – Portfolio

This is my personal portfolio website built using React to showcase my projects, skills, and resume.

## Live Website
https://tishamovaliya.github.io/my-portfolio/

## About
The portfolio includes:
- Introduction and professional summary
- Technical skills section
- Project showcase with live links
- Resume download
- Contact section

The website is fully responsive and optimized for both desktop and mobile devices.

## Tech Stack
- React
- JavaScript
- HTML
- CSS
- GitHub Pages (Deployment)

## Run Locally

Clone the repository:

git clone https://github.com/tishamovaliya/my-portfolio.git

Install dependencies:

npm install

Start development server:

npm start

## Deployment

Deployed using GitHub Pages.

To deploy updates:

npm run deploy
